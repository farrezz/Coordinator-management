Coordinator Management — Malmö 16

An internal coordination tool built for our team to manage weekly caseloads and schedules in real time.

## Features

- **Case tracking** — Log and track cases by category across each weekday, with weekly goals and progress
- **Decision overview** — Visual calendar showing upcoming decision deadlines with urgency indicators
- **Team schedule** — View and update absence for the whole team (vacation, sick leave, VAB, and more)
- **Absence periods** — Mark multi-week vacation in one step without filling in day by day
- **Live sync** — All changes are reflected instantly across every browser via Firebase
- **Dark / light mode** — Personal preference, saved per device

## Stack

- React + Vite
- Firebase Firestore (real-time database)
- Firebase Storage (background images)
- Deployed on Vercel

## Usage

This tool is intended for internal team use only. Access requires the team URL.
Admin functions (managing groups, tasks, roles, and themes) are PIN-protected.
